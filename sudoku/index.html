Constraint method

Classically sudokus are solved using constaints to identify a solution, the harder the sudoku the more reasoning is required in the constraints. Each colour represents an iteration of determing values from the particular set of constraints.

Simple uses the most underlying constraints based off of the rules of sudoku:

  1. Values must lie between 1 to 9
  2. A value can not be repeated in a column, row or box
